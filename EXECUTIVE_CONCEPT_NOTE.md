# Executive Concept Note

## Child Support Fraud & Integrity Scorecard (CSFIS)

The CSFIS prototype distinguishes among:
1. True fraud
2. Behavioral noncompliance
3. Structural burden

A central enhancement is the integration of **CP-NCP living distance**. Distance is not treated as fraud by itself. Instead, it is used to:
- improve fairness in interpretation,
- identify travel-related burden,
- support interstate case analysis,
- reduce false classification of low-income long-distance cases.

This package is suitable for:
- executive demonstrations,
- framework review,
- mock dashboard presentations,
- GitHub-to-Streamlit deployment,
- follow-on integration planning with agency systems.
